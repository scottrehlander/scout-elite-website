A Clip Project is your workspace for breaking down game footage into meaningful clips. It's where you identify key moments, annotate them, and prepare content for team review.

> **This feature requires a Solo subscription.** [Learn about our plans](/help/subscription-plans/).

## Creating a New Clip Project

1. Navigate to **Clip Creator** in the sidebar
2. Click **Create Clip Project**
3. Give your project a name (e.g., "vs Eagles - March 15 Game Review")
4. Select the videos you want to include in this project
5. Click **Create**

You'll be taken to the Clip Project workspace where you can start creating clips.

## The Clip Project Workspace

The workspace is the most feature-rich page in Scout Elite. It includes:

- **Video player** with custom playback controls
- **Timeline bar** showing all clips created for the current video
- **Clip creation tools** for setting in/out points
- **Tag editor** for annotating clips with sport-specific tags
- **Telestration tools** for drawing annotations on the video frame

## Adding Videos to a Project

A single Clip Project can contain multiple videos. This is useful when you have a game split across multiple recordings (e.g., one file per period).

To add more videos after creation:
1. Open the Clip Project
2. Use the video management panel to add videos from your Video Library

## Navigating Between Videos

Use the video selector in the workspace to switch between videos within the same project. Each video has its own timeline showing the clips you've created.
