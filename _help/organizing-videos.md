The Video Library is where all your uploaded footage lives. Keeping it organized with folders makes it easy to find the right video when you need it.

## Creating Folders

1. Go to **Video Library** in the sidebar
2. Click the **New Folder** button
3. Give your folder a descriptive name (e.g., "2025-26 Season", "Tournament Games", "Practice Footage")
4. Click **Create**

You can create folders inside folders to build a hierarchy that works for your team.

## Suggested Folder Structures

Here are some popular ways coaches organize their libraries:

**By season and event:**
```
2025-26 Season/
  ├── Regular Season/
  ├── Tournaments/
  └── Practice/
```

**By opponent:**
```
Games/
  ├── vs Eagles - Mar 15/
  ├── vs Hawks - Mar 22/
  └── vs Wolves - Mar 29/
```

**By player (for individual development):**
```
Player Development/
  ├── Alex - Skating/
  ├── Jordan - Shooting/
  └── Team Drills/
```

## Moving Videos Between Folders

Navigate to the video you want to move, use the video's context menu, and select the destination folder.

## Thumbnails

Each video gets an auto-generated thumbnail so you can quickly identify footage at a glance. Thumbnails are created automatically a few seconds after upload.

## AI Detection Indicator

If you're on the Pro plan and have run AI clip detection on videos within a folder, you'll see a lightning bolt (⚡) icon on that folder. This helps you quickly identify which footage has been analyzed by AI.
