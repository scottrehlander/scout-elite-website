Scout Elite Live is a free mobile companion app that lets you track in-game events in real time. Tap buttons to log goals, shots, penalties, and other events as they happen, then publish the game to Scout Elite for analysis.

## How It Works

1. **During the game** — open Scout Elite Live on your phone or tablet and tap event buttons as plays happen
2. **After the game** — publish the game events from the mobile app
3. **In Scout Elite** — view the published game in the **Scout Elite Live** section of the web app

## What Gets Tracked

Each event you log captures:

- **Event type** (goal, shot, penalty, faceoff, etc.)
- **Timestamp** (when during the game it happened)
- **Team** (which team the event belongs to)
- **Additional details** (player, notes, custom fields from the app)

## Viewing Published Games

1. Navigate to **Scout Elite Live** in the sidebar
2. You'll see a list of all published games (yours and any shared with you)
3. Click a game to view its details

## Game Report Tabs

Each published game has three views:

### Report Tab
A summary of the game including:
- Team statistics (total shots, goals, penalties, etc.)
- Scoring log
- Player-level stats

### Analytics Tab
Interactive charts showing event frequency over game time. Use these to identify momentum swings, pressure periods, and patterns.

### Log Tab
A filterable table of every event recorded during the game. You can edit events inline if you need to correct timestamps or details.

## Sharing Games

On the Pro plan, you can share published games with individuals or teams. This is great for distributing game reports to parents and assistant coaches. See [Sharing Videos, Clips & Reviews](/help/sharing-content/) for details.

## Connecting Live Events to Clips

One of the most powerful features of Scout Elite Live is the ability to import game events into a Clip Project. This lets you automatically create clips at the timestamps where events occurred — saving hours of manual clipping.

> **Tip:** Have a parent or assistant use Scout Elite Live during games. After the game, publish and share the events with the coach. The coach can then import them into a Clip Project to instantly create timestamped clips.
